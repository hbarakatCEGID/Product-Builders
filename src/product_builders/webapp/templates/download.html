Install the CLI

Product Builders is distributed as a Python package. Use pip from a checkout or your internal index.

From this repository (development)

One install includes the CLI, documentation server, tree-sitter AST analysis, and dev tools.

cd Product-Builders
pip install -e .
product-builders --help

Run this documentation server locally

python -m product_builders.webapp --reload

Then open http://127.0.0.1:8000 (default).

Typical workflow

  1. product-builders analyze <repo> --name <product>
  2. product-builders generate --name <product>
  3. Commit generated .cursor rules and governance under your product profile.
